# Contact Management System (Python)

A simple Python project to manage contacts using file handling.  
This project allows users to add, view, search, and delete contact information without using a database.

## Features
- Add new contact with name, phone number, and email
- View all saved contacts
- Search for a contact by name
- Delete a contact
- File-based storage (contacts.txt)

## Technologies Used
- Python
- File Handling
- Functions and Loops
